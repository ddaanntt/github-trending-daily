# 🔥 GitHub Trending Daily

每天自动收集 GitHub 上最热门的开源项目，帮你发现有趣、有潜力的开源世界。

## 📅 更新频率

每天早上 10:00（北京时间）自动更新。

## 📂 内容结构

```
content/
├── 2026-05-26.md    # 第1期
├── 2026-05-27.md    # 第2期
├── 2026-05-28.md    # 第3期
└── ...
```

## 🎯 筛选标准

- **热门项目**：当日 star 增速异常高（日增 200+）
- **潜力项目**：总 star 不高但增速快，有创新性
- **去重机制**：7 天内不重复推荐同一项目
- **覆盖范围**：全品类（前端、后端、AI、工具、游戏、设计...）

## 📊 数据来源

- [GitHub Trending](https://github.com/trending)
- [OSS Insight](https://ossinsight.io/)

## 🤖 自动化

本仓库由 [Hermes Agent](https://github.com/NousResearch/hermes-agent) 自动维护，每天定时抓取、筛选、生成内容并推送。
